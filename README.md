# Termux-Style
------------------
configura el estilo de colores y de letras en TERMUX, instalando el apk y sustituyendo el archivo de colores por defecto

primero instalar el apk de termux style

luego entrar a TERMUX y estar en la carpeta HOME, y en (MORE) entrar en STYLE y en CHOOSE COLOR presionar, escoger cualquier tema de colores
y seleccionarlo.

copiar o sustituir el archivo colors.properties que se encuentra en la carpeta .termux para cambiar el tipo de colores que estan en hexadecimal

ruta: ~/.termux
